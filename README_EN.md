## What This Is

**LXTI Travel Archetypes** is China's original MBTI-style travel assessment system.

Not astrology like "You're Pisces so you belong at the beach." It's based on **L/E Social, X/T Pace, S/I Experience, I/H Comfort** — four dimensions, **8-20 forced-choice questions**, pinpointing your travel DNA.

### 16 Archetypes. Zero Overlap. Zero Confusion.

| Type | Archetype Name | Core Profile |
|:----:|----------------|--------------|
| LXSI | Wilderness Solo Recharger | Alone, nature, slow walks, slow photos |
| LXII | Ancient Village Loafer | Escape the KPIs, tea in old towns, sunbathing |
| EXII | Street-Strolling Local | Follow your nose, pretend to live in another city |
| ETII | Alley-Hopping Foodie | Human GPS for food, mouth never stops |

> **All 16 archetype names, suffixes, and core profiles are completely unique — no repeated words, no similar associations, SBTI-level strong differentiation.**

### Four-in-One闭环

Not just a personality test. Final output:

```
📊 Personality Report (90% fixed standard + 10% personalized tweak)
  ↓
🎨 Exclusive Social Avatar (Q-style quirky comic, WeChat avatar standard)
  ↓
📍 Custom Travel Guide (Strictly matched to personality, no generic templates)
```

### Dynamic Adaptive Questioning

- Complete info → Streamlined 8-10 questions
- Medium info → Standard 12-15 questions
- Minimal info → Full 16-20 questions

**Every question must contribute to the profile. No waste.**

### Lazy Fast-Forward Channel

From question 4 onward, every question includes: "🚀 Can't be bothered, just give me the result"

AI infers personality from collected answers, uses reasonable defaults for unanswered dimensions. Results marked "⚡ Quick Assessment Edition", can be revised anytime.

## Technical Highlights

- ✅ **Original Four-Dimension System**: L/E · X/T · S/I · I/H, permanently locked, unmodifiable
- ✅ **Forced Mutual-Exclusion Questions**: All core questions use non-compatible options, forcing real trade-offs
- ✅ **90% Fixed Standard**: Same archetype, all users get identical core definitions, traits, preferences, pros/cons — guaranteed consistency
- ✅ **Dual-Channel Avatars**: With drawing capability → AI real-time generation; Without → Load from 16 local PNG fallbacks
- ✅ **Nationwide Universal**: Not bound to any destination, adaptable to all travel scenarios across China
- ✅ **Reserved MCP Ports**: Ready for seamless integration with travel resource databases, user preference storage, real-time travel data
