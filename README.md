## 为什么你的旅行总踩雷？

- 跟着网红攻略走，结果人挤人、排队三小时、拍照五分钟
- 明明想去放松，行程却比上班还累

- 去了「必打卡」景点，发现根本不是你的菜

**真相：不是攻略不好，是你根本不了解自己的旅行人格。**

---

## 这是什么

**LXTI 旅行人格** 是中国原创的 MBTI 式旅行测评体系。

不是「你是双鱼座所以适合海边」这种玄学。而是基于 **L/E 社交、X/T 节奏、S/I 体验、I/H 舒适** 四大维度，通过 **8-20 道强制互斥题目**，精准定位你的旅行 DNA。

### 16 型人格，零雷同、零混淆

| 类型 | 人格名称 | 核心画像 |
|:----:|---------|---------|
| LXSI | 旷野充电独行侠 | 一个人，大自然，慢慢走，慢慢拍 |
| LXII | 古村躺平摆烂仙 | 逃离KPI，古镇喝茶晒太阳 |
| EXII | 市井闲逛烟火民 | 跟着鼻子和直觉，假装在另一个城市生活 |
| ETII | 走街串巷逛吃控 | 美食GPS定位器，扫街嘴永不停歇 |
| ETIH | 人文带队全能王 | 朋友圈最强领队，有他在旅行不用带脑子 |
| ... | ... | ... |

> **全部 16 型人格名称、后缀、核心人设完全唯一，无重复字、无相似联想，达到 SBTI 级别的强区分度。**

### 四合一闭环

不只是人格测试，最终输出：

```
📊 人格报告（90%固定标准释义 + 10%个性化微调）
  ↓
🎨 专属社交人设头像（Q版搞怪漫画风，微信头像标准）
  ↓
📍 定制旅行攻略（严格贴合人格偏好，禁止通用模板）
```

### 动态自适应出题

- 信息完整 → 精简 8-10 题
- 信息中等 → 标准 12-15 题
- 信息极少 → 全面 16-20 题

**每道题都必须用于补全画像，无废题。**

### 懒人快进通道

从第 4 题开始，每道题都可选：「🚀 懒得回答了，直接给我结果」

AI 基于已收集的答案进行人格推断，对未回答维度采用合理默认。结果标注「⚡ 快速判定版」，可随时补充修正。

## 真实测评流程

```
用户："我想去成都玩，不知道该怎么安排"

AI："先做个 LXTI 旅行人格测评，帮你找到最适合的玩法！"

[一题一发，互斥选项，动态追踪]

Q1: 你更偏向哪种出行方式？
A. 一个人说走就走  B. 和好友/伴侣一起  C. 三五成群  D. 视旅伴而定

Q2: 一整天的旅行结束后，你更想：
A. 复盘照片，享受一个人的安静  B. 和朋友聊今天的经历  C. 和新朋友继续热闹

...

[测评完成]

🎉 你的人格是：ETII｜走街串巷逛吃控

> 你的旅行驱动力简单粗暴——吃。每一座城市的终极吸引力在于街头巷尾的味道。
> 行程以"一日多餐"为骨架，景点仅作消食穿插。

🍴 成都 3 日攻略（走街串巷逛吃控专属）：
- Day 1: 魁星楼街早午餐 → 人民公园鹤鸣茶社 → 奎星楼街夜市
- Day 2: 青羊小区菜市场 → 玉林综合市场 → 九眼桥小酒馆
- Day 3: sleeping in → 查漏补缺清单 → 返程

⚠️ 避雷：景区内的"成都小吃套餐"、商场连锁火锅、纯拍照不吃饭
```

## 技术亮点

- ✅ **原创四维体系**：L/E · X/T · S/I · I/H，永久锁死，不可修改
- ✅ **强制互斥出题**：所有核心题目采用不可兼得选项，逼迫真实取舍
- ✅ **90% 固定标准**：同一人格，所有用户核心定义、性格、偏好、优缺点完全统一，保证公信力
- ✅ **双通道头像**：有绘图能力→AI实时生成；无绘图能力→读取本地16张预存PNG兜底
- ✅ **全国通用**：不绑定任何目的地，适配全国所有文旅场景
- ✅ **预留 MCP 接口**：可无缝对接文旅资源库、用户偏好存储、实时出行数据
